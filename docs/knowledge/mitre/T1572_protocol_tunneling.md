# T1572 - 协议隧道

## 技术信息
- **技术ID**: T1572
- **技术名称**: Protocol Tunneling
- **战术分类**: Command and Control (C2)

## DGA关联性
DGA恶意软件可能使用DNS隧道技术，将C2数据编码在DNS查询和响应中。通过DGA生成的域名作为DNS隧道的载体，实现隐蔽的数据传输。

## 检测方法
- 检测DNS查询中的异常编码数据
- 监控DNS查询长度异常
- 分析DNS TXT记录的数据量
- 检测DNS查询频率与数据传输量的关联

## 示例指标
- DNS查询的子域名部分包含Base64/Hex编码数据
- 单个域名的DNS查询频率异常高
- DNS TXT记录响应数据量大
- DNS查询模式与正常浏览行为不符
