# T1105 - 入站工具传输 (Ingress Tool Transfer)

## ATT&CK 映射
- **战术**: 命令与控制 (Command and Control)
- **技术 ID**: T1105

## 与 DGA 的关联
恶意软件通过 DGA 域名解析到 C2 服务器后，常从该服务器下载额外的攻击工具、载荷更新或配置文件。DGA 确保了下载通道的持续可用性。

## 检测策略
- 监控通过新解析域名下载可执行文件的行为
- 检测 DNS 解析后立即发生的大数据量传输
- 分析下载文件的类型、大小和频率异常
- 关注从低信誉域名下载 PE/ELF 等可执行格式

## 响应措施
- 在代理层面阻断从未分类域名下载可执行文件
- 对下载内容实施沙箱检测
- 记录并分析所有通过 DGA 域名传输的文件哈希
- 与端点检测响应（EDR）联动阻断恶意载荷执行
