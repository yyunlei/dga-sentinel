# T1568.002 - 动态解析：域名生成算法

## 技术信息
- **技术ID**: T1568.002
- **技术名称**: Dynamic Resolution: Domain Generation Algorithms
- **战术分类**: Command and Control (C2)

## DGA关联性
这是DGA技术的直接对应MITRE条目。恶意软件使用算法动态生成大量域名，攻击者只需注册其中少量域名即可建立C2通信，使防御方难以通过黑名单方式阻断。

## 检测方法
- 部署DGA检测模型（机器学习/深度学习）
- 监控NXDomain响应比率
- 分析域名字符分布和熵值
- 检测域名注册时间与查询时间的关联
- 使用已知DGA家族的逆向算法进行预测

## 示例指标
- 域名字符随机性高于正常域名
- 短时间内查询大量不存在的域名
- 域名注册时间与首次查询时间接近
- 域名长度和字符集符合已知DGA模式
