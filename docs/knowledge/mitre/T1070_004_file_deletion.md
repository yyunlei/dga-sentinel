# T1070.004 - 指标清除：文件删除

## 技术信息
- **技术ID**: T1070.004
- **技术名称**: Indicator Removal: File Deletion
- **战术分类**: Defense Evasion（防御规避）

## DGA关联性
DGA恶意软件在执行后常删除自身文件和日志以清除痕迹。DGA配置文件、种子文件等关键证据可能被主动清除。

## 检测方法
- 监控可疑文件删除操作
- 启用文件系统审计日志
- 检测日志清除行为
- 使用EDR工具记录文件操作历史

## 示例指标
- 恶意进程执行后自删除
- 系统日志被清除或截断
- 临时文件夹中的可疑文件删除
- 与DGA活动时间相关的文件删除事件
