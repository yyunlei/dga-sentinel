# T1071.004 - 应用层协议：DNS

## 技术信息
- **技术ID**: T1071.004
- **技术名称**: Application Layer Protocol: DNS
- **战术分类**: Command and Control (C2)

## DGA关联性
DGA域名通过DNS协议进行解析，是DGA恶意软件最核心的通信方式。攻击者利用DNS协议的普遍性和通常不被严格监控的特点，将C2指令编码在DNS查询和响应中。

## 检测方法
- 监控DNS查询频率和模式异常
- 分析DNS查询的域名熵值
- 检测大量NXDomain响应
- 监控DNS TXT记录查询异常
- 使用被动DNS数据进行关联分析

## 示例指标
- 单主机短时间内大量DNS查询
- DNS查询域名的二级域名随机性高
- DNS响应中包含异常编码数据
- 查询不常见TLD的域名
