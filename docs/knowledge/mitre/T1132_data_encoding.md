# T1132 - 数据编码

## 技术信息
- **技术ID**: T1132
- **技术名称**: Data Encoding
- **战术分类**: Command and Control (C2)

## DGA关联性
DGA恶意软件在C2通信中广泛使用数据编码技术。域名本身可能包含编码信息，C2通信数据也常使用Base64、XOR、自定义编码方案。

## 检测方法
- 检测DNS查询中的Base64/Hex编码模式
- 分析HTTP请求体中的编码数据
- 监控非标准编码方案的使用
- 使用熵值分析识别编码数据

## 示例指标
- DNS子域名包含Base64字符集
- HTTP POST数据熵值异常高
- 通信数据包含可识别的编码头部
- 固定长度的编码数据块
