# T1568 - 动态解析 (Dynamic Resolution)

## ATT&CK 映射
- **战术**: 命令与控制 (Command and Control)
- **技术 ID**: T1568
- **子技术**: T1568.002 域名生成算法 (DGA)

## 与 DGA 的关联
这是 DGA 最直接对应的 ATT&CK 技术。攻击者使用算法动态生成大量候选域名，仅需注册其中少量即可建立 C2 通信，极大增加了防御方的封锁难度。

## 检测策略
- 部署 DGA 检测模型（机器学习/深度学习）
- 监控单主机短时间内大量 NXDOMAIN 响应
- 分析域名字符分布、长度、熵值等统计特征
- 与威胁情报中已知 DGA 域名列表进行匹配

## 响应措施
- 实施 DNS Sinkhole 将 DGA 域名导向受控服务器
- 利用已知算法预测并预注册/封锁未来域名
- 隔离产生大量 DGA 查询的受感染主机
- 向威胁情报平台共享新发现的 DGA 模式
