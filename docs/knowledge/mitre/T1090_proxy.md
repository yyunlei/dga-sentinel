# T1090 - 代理

## 技术信息
- **技术ID**: T1090
- **技术名称**: Proxy
- **战术分类**: Command and Control (C2)

## DGA关联性
部分DGA恶意软件使用多层代理架构隐藏真实C2服务器。DGA域名可能解析到代理节点而非最终C2，增加追踪难度。Fast-Flux DNS技术也属于此类。

## 检测方法
- 检测DGA域名解析IP的频繁变化（Fast-Flux）
- 监控已知代理服务的异常使用
- 分析网络流量中的多跳通信模式
- 检测Tor/I2P等匿名网络连接

## 示例指标
- 同一域名短时间内解析到多个不同IP
- 与已知Tor出口节点的连接
- 多层代理跳转的网络流量
- 域名TTL值异常低
