# T1001 - 数据混淆

## 技术信息
- **技术ID**: T1001
- **技术名称**: Data Obfuscation
- **战术分类**: Command and Control (C2)

## DGA关联性
DGA恶意软件使用多种数据混淆技术隐藏C2通信内容。包括在正常流量中嵌入C2数据、使用垃圾数据填充、模拟合法协议格式等。

## 检测方法
- 分析网络流量中的统计异常
- 检测协议格式的细微偏差
- 使用机器学习识别混淆流量
- 对比正常流量基线

## 示例指标
- HTTP请求头部包含异常字段
- 数据包大小分布与正常流量不同
- 通信时间间隔过于规律
- 协议字段值超出正常范围
