# T1583.001 - 获取基础设施：域名

## 技术信息
- **技术ID**: T1583.001
- **技术名称**: Acquire Infrastructure: Domains
- **战术分类**: Resource Development（资源开发）

## DGA关联性
攻击者需要注册DGA算法生成的域名才能建立C2通信。通常只注册少量域名，利用DGA的大量候选域名增加防御方预测难度。

## 检测方法
- 监控新注册域名与DGA模式的匹配
- 分析域名注册信息（WHOIS）的异常
- 检测批量域名注册行为
- 使用被动DNS数据追踪域名注册趋势

## 示例指标
- 新注册域名符合已知DGA模式
- 域名注册使用隐私保护服务
- 批量注册的域名具有相似特征
- 域名注册后短时间内出现DNS查询
