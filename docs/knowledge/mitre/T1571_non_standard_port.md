# T1571 - 非标准端口

## 技术信息
- **技术ID**: T1571
- **技术名称**: Non-Standard Port
- **战术分类**: Command and Control (C2)

## DGA关联性
DGA恶意软件在解析域名获取C2 IP后，可能使用非标准端口进行通信。例如在高端口上运行HTTP协议，或使用非53端口的DNS通信。

## 检测方法
- 监控非标准端口的HTTP/HTTPS流量
- 检测高端口上的持续连接
- 分析端口使用模式异常
- 关联DGA域名解析与非标准端口连接

## 示例指标
- DGA域名解析IP的非80/443端口连接
- 高端口上的HTTP协议通信
- 非53端口的DNS查询
- 与已知恶意端口的连接
