# T1008 - 备用信道 (Fallback Channels)

## ATT&CK 映射
- **战术**: 命令与控制 (Command and Control)
- **技术 ID**: T1008

## 与 DGA 的关联
DGA 本身就是一种备用信道机制。当主要 C2 域名被封锁时，恶意软件通过 DGA 生成新的候选域名，确保与攻击者的通信不会中断。部分恶意软件同时使用硬编码域名和 DGA 作为双重保障。

## 检测策略
- 监控主机在主域名连接失败后的 DNS 查询行为变化
- 检测短时间内对大量不同域名的连续查询
- 关注从硬编码域名切换到随机域名的行为模式
- 分析 DNS 查询失败后的重试策略和时间间隔

## 响应措施
- 同时封锁硬编码 C2 域名和 DGA 生成域名
- 部署行为分析引擎检测信道切换行为
- 在端点层面监控恶意软件的网络重连逻辑
- 建立域名封锁的自动化更新机制
