# T1219 - 远程访问软件

## 技术信息
- **技术ID**: T1219
- **技术名称**: Remote Access Software
- **战术分类**: Command and Control (C2)

## DGA关联性
部分DGA恶意软件在建立C2通信后，下载并部署远程访问工具（RAT）。DGA域名用于初始C2通信，后续通过RAT实现持久化远程控制。

## 检测方法
- 监控DGA域名解析后的RAT下载行为
- 检测已知RAT工具的网络特征
- 分析远程桌面协议的异常使用
- 关联DGA告警与RAT部署事件

## 示例指标
- DGA域名解析后下载VNC/TeamViewer等工具
- 非授权的远程桌面连接
- RAT特征的网络流量
- 异常的屏幕共享或键盘记录行为
