# Fobber DGA 家族

## 基本信息
- **家族名称**: Fobber
- **别名**: 无
- **首次发现**: 2015年
- **算法类型**: 哈希型

## 域名生成特征
- **域名长度**: 12-16个字符
- **使用TLD**: .com, .net
- **生成模式**: 基于日期和硬编码密钥的哈希运算
- **日生成量**: 约200个域名/天

## 关联恶意软件
- Fobber银行木马（Tinba变种）
- 针对欧洲银行客户

## C2基础设施
- 使用DGA作为备用C2机制
- 主C2为硬编码地址

## 检测指标
- 域名字符分布接近均匀随机
- 域名熵值较高
- 与网银访问行为时间相关
- 注入浏览器进程的行为
