# Qadars DGA 家族

## 基本信息
- **家族名称**: Qadars
- **别名**: 无
- **首次发现**: 2013年
- **算法类型**: 哈希型

## 域名生成特征
- **域名长度**: 12-16个字符
- **使用TLD**: .com, .net
- **生成模式**: 基于SHA256哈希截取
- **日生成量**: 约200个域名/天

## 关联恶意软件
- Qadars银行木马
- 针对荷兰、法国银行客户

## C2基础设施
- 使用DGA和硬编码C2双重机制
- 通过Tor网络进行部分通信

## 检测指标
- 域名字符分布均匀
- 域名熵值极高
- 与欧洲银行网站访问相关
- Web注入攻击行为
