# Bedep DGA 家族

## 基本信息
- **家族名称**: Bedep
- **别名**: 无
- **首次发现**: 2015年
- **算法类型**: 算术运算型

## 域名生成特征
- **域名长度**: 12-16个字符
- **使用TLD**: .com, .net, .org
- **生成模式**: 基于日期和硬编码常量的算术运算
- **日生成量**: 约200个域名/天

## 关联恶意软件
- Bedep广告欺诈木马
- 常通过Angler漏洞利用工具包分发

## C2基础设施
- 使用多层代理隐藏真实C2
- 域名注册分散在多个注册商

## 检测指标
- 域名由小写字母组成，无数字
- 域名长度分布集中
- 与Angler EK流量共现
- DNS查询时间与系统日期强相关
