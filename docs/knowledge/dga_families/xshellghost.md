# XshellGhost DGA 家族

## 基本信息
- **家族名称**: XshellGhost
- **别名**: 无
- **首次发现**: 2017年
- **算法类型**: 哈希型

## 域名生成特征
- **域名长度**: 16-20个字符
- **使用TLD**: .com
- **生成模式**: 基于日期和硬编码密钥的哈希运算
- **日生成量**: 约1个域名/月（低频率）

## 关联恶意软件
- Xshell后门（供应链攻击）
- NetSarang软件被植入后门

## C2基础设施
- 通过DNS TXT记录传递加密指令
- 极低频率的DGA避免检测

## 检测指标
- DNS TXT记录查询异常
- 与NetSarang软件更新相关
- 域名生成频率极低
- 供应链攻击特征
