# Murofet

## 基本信息
- **算法类型**: 基于时间的伪随机算法 (Time-based PRNG)
- **域名长度**: 12-22 个字符
- **TLD**: .com, .net, .org, .info, .biz
- **每日生成量**: 约 800 个域名

## 特征
- 域名由纯小写字母组成，无数字或特殊字符
- 使用日期和硬编码常量作为种子
- 域名字符分布呈现非自然语言的统计特征

## 关联恶意软件
Murofet 木马，与 Zeus/Zbot 银行木马家族密切相关，用于窃取金融凭证

## 检测建议
- 利用字符 n-gram 频率偏离自然语言的程度进行检测
- 监控对多个主流 TLD 的并发查询行为
- 与 Zeus 家族 IoC 进行关联分析
- 纯字母域名需结合长度和熵值综合判断
