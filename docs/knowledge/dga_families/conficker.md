# Conficker

## 基本信息
- **算法类型**: 基于时间的伪随机算法 (Time-based PRNG)
- **域名长度**: 5-11 个字符
- **TLD**: .com, .net, .org, .info, .biz 等
- **每日生成量**: 250-50000 个域名（因变种而异）

## 特征
- 域名由纯小写字母组成，无数字或连字符
- 使用当前日期作为种子，每日更新域名列表
- 变种 A/B/C 算法复杂度递增，C 变种每日生成 50000 个域名

## 关联恶意软件
Conficker 蠕虫（又名 Downadup/Kido），利用 MS08-067 漏洞传播

## 检测建议
- 监控短域名 + 高熵值组合的 DNS 请求
- 关注单主机短时间内大量 NXDOMAIN 响应
- 利用已知种子算法进行预测性阻断
