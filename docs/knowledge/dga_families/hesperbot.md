# HesperBot DGA 家族

## 基本信息
- **家族名称**: HesperBot
- **别名**: Win32/Spy.Hesperbot
- **首次发现**: 2013年
- **算法类型**: 算术运算型

## 域名生成特征
- **域名长度**: 10-15个字符
- **使用TLD**: .com, .net
- **生成模式**: 基于日期和常量的模运算
- **日生成量**: 约100个域名/天

## 关联恶意软件
- HesperBot银行木马
- 针对土耳其、捷克、葡萄牙银行

## C2基础设施
- 通过钓鱼邮件传播
- C2服务器位于东欧

## 检测指标
- 域名由小写字母组成
