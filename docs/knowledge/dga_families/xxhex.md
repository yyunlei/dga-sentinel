# Xxhex DGA 家族

## 基本信息
- **家族名称**: Xxhex
- **别名**: 无
- **首次发现**: 2015年
- **算法类型**: 十六进制哈希型

## 域名生成特征
- **域名长度**: 16-32个字符（十六进制）
- **使用TLD**: .com, .net
- **生成模式**: 基于日期的哈希运算，输出十六进制字符串
- **日生成量**: 约100个域名/天

## 关联恶意软件
- Xxhex后门程序
- 用于数据窃取

## C2基础设施
- 使用DGA作为主要C2通信
- 域名全部为十六进制字符

## 检测指标
- 域名仅包含0-9和a-f字符
- 域名看起来像哈希值
- 域名熵值极高
- 正则匹配 `^[0-9a-f]{16,32}$`
