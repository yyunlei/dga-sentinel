# Madmax DGA 家族

## 基本信息
- **家族名称**: Madmax
- **别名**: 无
- **首次发现**: 2016年
- **算法类型**: 词表型

## 域名生成特征
- **域名长度**: 12-25个字符
- **使用TLD**: .com, .org, .net
- **生成模式**: 从内置英语词表中随机组合2-3个单词
- **日生成量**: 约300个域名/天

## 关联恶意软件
- Madmax后门程序
- 用于数据窃取和远程控制

## C2基础设施
- 使用CDN服务隐藏真实IP
- 域名注册使用隐私保护

## 检测指标
- 域名由可读英语单词组成
- 域名较长但看似合法
- 需要NLP分析判断语义合理性
- 域名注册时间集中
