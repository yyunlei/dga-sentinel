# Tempedreve DGA 家族

## 基本信息
- **家族名称**: Tempedreve
- **别名**: 无
- **首次发现**: 2015年
- **算法类型**: 算术运算型

## 域名生成特征
- **域名长度**: 8-12个字符
- **使用TLD**: .com, .net
- **生成模式**: 基于日期的简单算术运算
- **日生成量**: 约50个域名/天

## 关联恶意软件
- Tempedreve蠕虫
- 通过可移动存储设备传播

## C2基础设施
- 简单的HTTP C2通信
- 使用DGA获取更新指令
