# Vawtrak DGA 家族

## 基本信息
- **家族名称**: Vawtrak
- **别名**: Neverquest, Snifula
- **首次发现**: 2013年
- **算法类型**: 算术运算型

## 域名生成特征
- **域名长度**: 10-16个字符
- **使用TLD**: .com, .net, .org
- **生成模式**: 基于日期和配置参数的算术运算
- **日生成量**: 约200个域名/天

## 关联恶意软件
- Vawtrak银行木马
- 针对全球银行和电商网站

## C2基础设施
- 使用Tor2Web代理进行C2通信
- DGA作为备用通信渠道

## 检测指标
- 域名字符随机性中等
- 与Tor2Web代理的连接
- Web注入攻击行为
- 凭证窃取和屏幕截图
