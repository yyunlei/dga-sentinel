# Qakbot DGA 家族

## 基本信息
- **家族名称**: Qakbot
- **别名**: QBot, QuakBot, Pinkslipbot
- **首次发现**: 2008年（DGA版本约2016年）
- **算法类型**: 算术运算型

## 域名生成特征
- **域名长度**: 8-25个字符
- **使用TLD**: .com, .net, .org, .info, .biz
- **生成模式**: 基于日期和配置种子的算术运算
- **日生成量**: 约500个域名/天

## 关联恶意软件
- Qakbot银行木马/加载器
- 分发ProLock、Egregor勒索软件
- 2023年FBI摧毁后再次复活

## C2基础设施
- 多层代理网络
- 使用被入侵主机作为中间代理
- DGA作为备用通信渠道

## 检测指标
- 域名字符随机性高
- HTTPS通信使用自签名证书
- 横向移动行为
- 与勒索软件活动关联
