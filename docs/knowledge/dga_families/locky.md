# Locky DGA 家族

## 基本信息
- **家族名称**: Locky
- **别名**: 无
- **首次发现**: 2016年
- **算法类型**: 哈希型 + 算术运算型

## 域名生成特征
- **域名长度**: 8-18个字符
- **使用TLD**: .ru, .info, .biz, .click, .work, .xyz
- **生成模式**: 基于硬编码种子、日期和配置参数
- **日生成量**: 约6-12个域名/天（多个种子）

## 关联恶意软件
- Locky勒索软件
- 通过Necurs僵尸网络分发
- 多个变种：Diablo6, Lukitus, Zepto

## C2基础设施
- DGA用于获取加密密钥和支付页面
- 与Necurs僵尸网络紧密关联

## 检测指标
- 使用非常规TLD
- 域名字符随机性高
- 伴随大量文件加密行为
- 与Necurs垃圾邮件活动共现
