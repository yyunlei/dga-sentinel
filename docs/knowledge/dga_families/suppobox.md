# Suppobox

## 基本信息
- **算法类型**: 基于词典的拼接算法 (Wordlist-based)
- **域名长度**: 12-24 个字符
- **TLD**: .com, .net
- **每日生成量**: 约 1024 个域名

## 特征
- 使用英文单词词典拼接生成域名，外观接近合法域名
- 域名通常由 2-3 个常见英文单词组合而成
- 对传统熵值检测方法具有较强的规避能力

## 关联恶意软件
Suppobox 木马，主要用于窃取银行凭证和敏感信息

## 检测建议
- 传统字符熵值检测效果有限，需结合词频分析
- 使用 NLP 模型检测非自然的单词组合模式
- 关注新注册域名中的多单词拼接模式
- 建议使用深度学习模型（如 LSTM/CNN）进行分类
