# Dyre

## 基本信息
- **算法类型**: 基于哈希的伪随机算法 (Hash-based PRNG)
- **域名长度**: 17-34 个字符
- **TLD**: .cc, .ws, .to, .so 等冷门 TLD
- **每日生成量**: 1000 个域名

## 特征
- 域名由十六进制字符（a-f, 0-9）组成，类似哈希值
- 使用 SHA-256 哈希结合日期种子生成
- 域名长度较长且字符集有限，特征明显

## 关联恶意软件
Dyre（又名 Dyreza）银行木马，专门针对企业网银进行中间人攻击

## 检测建议
- 检测仅包含十六进制字符的长域名
- 监控对冷门 TLD 的异常 DNS 查询
- 十六进制域名 + 非主流 TLD 组合是高置信度指标
- 建议在 DNS 防火墙中添加正则规则匹配
