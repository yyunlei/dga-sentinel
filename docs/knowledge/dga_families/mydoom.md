# Mydoom DGA 家族

## 基本信息
- **家族名称**: Mydoom
- **别名**: Novarg, Shimgapi
- **首次发现**: 2004年
- **算法类型**: 字符随机生成

## 域名生成特征
- **域名长度**: 5-10个字符
- **使用TLD**: .com
- **生成模式**: 随机字符组合，早期DGA实现
- **日生成量**: 较少，约20-50个域名/天

## 关联恶意软件
- Mydoom蠕虫
- 历史上传播速度最快的邮件蠕虫之一

## C2基础设施
- 早期使用简单的DGA机制
- 主要通过邮件传播

## 检测指标
- 域名短且随机
- 大量邮件发送行为
- 搜索引擎查询异常
- 后门端口（TCP 3127）开放
