# Tinba

## 基本信息
- **算法类型**: 基于种子的伪随机算法 (Seed-based PRNG)
- **域名长度**: 12-22 个字符
- **TLD**: .com
- **每日生成量**: 约 1000 个域名

## 特征
- 域名由小写字母和数字混合组成
- 使用硬编码种子结合线性同余生成器 (LCG)
- 仅使用 .com TLD，增加了与合法域名的混淆度

## 关联恶意软件
Tinba（Tiny Banker），体积仅约 20KB 的超小型银行木马，通过 Web 注入窃取银行凭证

## 检测建议
- 仅限 .com TLD 使得基于 TLD 的过滤无效
- 需依赖字符分布和 n-gram 模型进行检测
- 关注字母数字混合域名的查询频率异常
- 结合浏览器进程注入行为进行端点检测
