# Virut DGA 家族

## 基本信息
- **家族名称**: Virut
- **别名**: 无
- **首次发现**: 2006年
- **算法类型**: 算术运算型

## 域名生成特征
- **域名长度**: 6-10个字符
- **使用TLD**: .com
- **生成模式**: 基于简单算术运算的字符生成
- **日生成量**: 约10-20个域名/天

## 关联恶意软件
- Virut文件感染型病毒
- 感染PE文件和HTML文件

## C2基础设施
- 使用IRC协议进行C2通信
- DGA用于备用IRC服务器发现

## 检测指标
- 域名较短
- IRC通信流量
- PE文件被感染（入口点修改）
