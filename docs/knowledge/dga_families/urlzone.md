# URLZone DGA 家族

## 基本信息
- **家族名称**: URLZone
- **别名**: Bebloh, Shiotob
- **首次发现**: 2009年（DGA版本约2015年）
- **算法类型**: 词表型

## 域名生成特征
- **域名长度**: 10-20个字符
- **使用TLD**: .com, .net
- **生成模式**: 从内置词表中组合单词
- **日生成量**: 约100个域名/天

## 关联恶意软件
- URLZone银行木马
- 自动计算安全转账金额以逃避检测

## C2基础设施
- 使用DGA和硬编码C2双重机制
- 智能化的转账金额计算

## 检测指标
- 域名由英语单词片段组成
- 域名看似合法
- 异常的银行转账行为
- 转账金额被自动调整
