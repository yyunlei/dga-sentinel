# Emotet DGA 家族

## 基本信息
- **家族名称**: Emotet
- **别名**: Heodo, Geodo
- **首次发现**: 2014年
- **算法类型**: 词表型 + 算术运算型

## 域名生成特征
- **域名长度**: 8-24个字符
- **使用TLD**: .com, .net, .org
- **生成模式**: 从内置词表中组合单词片段
- **日生成量**: 变化较大，数百至数千

## 关联恶意软件
- Emotet银行木马/加载器
- 分发TrickBot、Ryuk勒索软件
- 全球最危险的僵尸网络之一

## C2基础设施
- 多层C2架构（Tier1/Tier2/Tier3）
- 使用被入侵的合法网站作为C2
- 2021年国际执法行动摧毁后多次复活

## 检测指标
- 域名包含类英语单词片段
- HTTP POST请求携带加密数据
- 大量垃圾邮件发送行为
- 与TrickBot/Ryuk活动关联
