# Shifu DGA 家族

## 基本信息
- **家族名称**: Shifu
- **别名**: 无
- **首次发现**: 2015年
- **算法类型**: 词表型

## 域名生成特征
- **域名长度**: 10-20个字符
- **使用TLD**: .com
- **生成模式**: 从内置词表中组合单词
- **日生成量**: 约50个域名/天

## 关联恶意软件
- Shifu银行木马
- 融合多个银行木马技术（Zeus、Gozi、Dridex）

## C2基础设施
- 使用DGA和硬编码C2双重机制
- 主要针对日本银行

## 检测指标
- 域名由英语单词组成
- 域名看似合法
- 与日本银行网站访问相关
- 使用多种反分析技术
