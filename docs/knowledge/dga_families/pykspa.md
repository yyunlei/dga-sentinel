# Pykspa

## 基本信息
- **算法类型**: 基于词典的拼接算法 (Wordlist-based)
- **域名长度**: 6-16 个字符
- **TLD**: .com, .net, .org, .info, .biz
- **每日生成量**: 约 800 个域名

## 特征
- 使用内置词典生成类似自然语言的域名
- 域名外观接近合法域名，具有较强隐蔽性
- 通过 Skype 等即时通讯工具传播

## 关联恶意软件
Pykspa 蠕虫，通过 Skype 消息传播，窃取用户信息并加入僵尸网络

## 检测建议
- 词典型 DGA 对传统熵值检测具有规避能力
- 需使用深度学习模型捕捉细微的语言模式异常
- 关注域名中常见单词的非常规组合方式
- 结合 DNS 查询时序模式进行行为分析
