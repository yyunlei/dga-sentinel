# Pushdo DGA 家族

## 基本信息
- **家族名称**: Pushdo
- **别名**: Pandex, Cutwail
- **首次发现**: 2007年（DGA版本约2014年）
- **算法类型**: 算术运算型

## 域名生成特征
- **域名长度**: 8-12个字符
- **使用TLD**: .com, .net, .kz
- **生成模式**: 基于日期和硬编码常量的运算
- **日生成量**: 约30个域名/天

## 关联恶意软件
- Pushdo下载器
- Cutwail垃圾邮件僵尸网络

## C2基础设施
- 多层架构：Pushdo下载Cutwail
- 使用DGA作为备用C2

## 检测指标
- 域名较短且字符随机
- 大量SMTP外发连接
- 垃圾邮件发送行为
- 与Cutwail流量特征共现
