# DNSChanger DGA 家族

## 基本信息
- **家族名称**: DNSChanger
- **别名**: Alureon DNS, RSPlug
- **首次发现**: 2007年
- **算法类型**: 算术运算型

## 域名生成特征
- **域名长度**: 8-15个字符
- **使用TLD**: .com, .net
- **生成模式**: 基于日期和固定种子的算术运算
- **日生成量**: 约50个域名/天

## 关联恶意软件
- DNSChanger木马
- 修改系统DNS设置劫持流量

## C2基础设施
- 使用恶意DNS服务器重定向流量
- 2011年FBI "Operation Ghost Click" 行动摧毁

## 检测指标
- 系统DNS设置被篡改
- DNS查询被重定向到非授权服务器
- 域名模式相对简单可预测
- 浏览器搜索结果被劫持
