# Sisron DGA 家族

## 基本信息
- **家族名称**: Sisron
- **别名**: 无
- **首次发现**: 2014年
- **算法类型**: 算术运算型

## 域名生成特征
- **域名长度**: 8-12个字符
- **使用TLD**: .com, .net
- **生成模式**: 基于简单算术运算和日期种子
- **日生成量**: 约100个域名/天

## 关联恶意软件
- Sisron后门程序
- 用于远程控制和数据窃取

## C2基础设施
- 简单的HTTP C2通信
- 使用DGA作为主要C2机制

## 检测指标
- 域名由小写字母组成
- 域名长度较短
- HTTP通信使用固定路径模式
- 数据外传行为
