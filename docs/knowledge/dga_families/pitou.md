# Pitou DGA 家族

## 基本信息
- **家族名称**: Pitou
- **别名**: 无
- **首次发现**: 2014年
- **算法类型**: 算术运算型

## 域名生成特征
- **域名长度**: 12-16个字符
- **使用TLD**: .com, .net, .org
- **生成模式**: 基于日期和系统信息的算术运算
- **日生成量**: 约300个域名/天

## 关联恶意软件
- Pitou Bootkit/Rootkit
- 内核级恶意软件，修改MBR

## C2基础设施
- 使用DGA获取垃圾邮件发送指令
- 作为垃圾邮件僵尸网络运作

## 检测指标
- 域名由小写字母组成
- MBR被修改的痕迹
- 大量SMTP外发连接
- 内核驱动异常加载
