# Ramnit

## 基本信息
- **算法类型**: 基于时间的伪随机算法 (Time-based PRNG)
- **域名长度**: 8-19 个字符
- **TLD**: .com, .eu, .bid
- **每日生成量**: 约 100 个域名

## 特征
- 域名由小写字母组成，偶尔包含数字
- 使用日期和硬编码种子生成，每日更新
- 生成量相对较少，增加了逐一注册的可行性

## 关联恶意软件
Ramnit 银行木马/蠕虫，通过感染可执行文件和 HTML 文件传播，窃取银行凭证和 FTP 密码

## 检测建议
- 每日生成量少，攻击者可能全部注册，需关注新注册域名
- 监控对 .eu 和 .bid 等非常规 TLD 的查询
- 结合文件感染行为指标进行主机侧关联分析
- 利用已知算法进行预测性域名封锁
