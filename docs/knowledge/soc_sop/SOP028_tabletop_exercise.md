# SOP028 - 桌面推演

## SOP信息
- **编号**: SOP028
- **标题**: DGA安全事件桌面推演流程
- **触发条件**: 定期执行（每半年）或重大安全事件后

## 响应步骤
1. 设计推演场景（大规模DGA爆发、APT攻击、勒索软件等）
2. 确定参与人员（SOC、IT运维、业务方、管理层）
3. 分发推演材料和角色分配
4. 按时间线推进场景，注入新事件
5. 各角色按SOP执行响应动作
6. 记录决策过程和响应时间
7. 推演结束后进行复盘讨论
8. 输出改进建议和行动计划

## 升级标准
- 推演中发现重大流程缺陷时制定紧急修复计划
- 推演结果需向管理层汇报

## 使用工具
- 推演脚本、白板/协作工具、计时器
