# SOP015 - 安全事件报告

## SOP信息
- **编号**: SOP015
- **标题**: DGA安全事件报告流程
- **触发条件**: DGA安全事件需要正式报告

## 响应步骤
1. 收集事件时间线（发现、响应、遏制、恢复）
2. 整理技术细节（DGA家族、IOC、影响范围）
3. 评估业务影响（数据泄露、服务中断）
4. 编写事件报告（包含执行摘要和技术详情）
5. 内部评审报告内容
6. 提交给安全主管和管理层
7. 根据合规要求决定是否外部报告

## 升级标准
- 涉及个人数据泄露时需在72小时内报告监管机构
- 涉及关键基础设施时按国家规定报告

## 使用工具
- 事件管理平台、报告模板、合规管理系统
