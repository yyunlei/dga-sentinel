# SOP014 - 防火墙规则创建

## SOP信息
- **编号**: SOP014
- **标题**: DGA相关防火墙规则创建流程
- **触发条件**: 需要在防火墙上阻断DGA相关流量

## 响应步骤
1. 确定阻断目标（IP地址、域名、端口）
2. 评估规则影响（是否影响正常业务）
3. 编写防火墙规则（包含源/目的/端口/协议）
4. 在测试环境验证规则
5. 获取变更审批
6. 在生产环境部署规则
7. 监控规则命中情况
8. 记录规则创建原因和有效期

## 升级标准
- 规则影响正常业务时立即回滚并升级
- 规则命中量异常大时通知安全主管

## 使用工具
- 防火墙管理平台、变更管理系统、SIEM
