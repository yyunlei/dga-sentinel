# SOP016 - 证据保全

## SOP信息
- **编号**: SOP016
- **标题**: DGA安全事件证据保全流程
- **触发条件**: DGA安全事件需要保全数字证据

## 响应步骤
1. 确定需要保全的证据类型（内存、磁盘、网络日志、DNS日志）
2. 使用写保护工具防止证据被修改
3. 采集内存镜像（使用Volatility/WinPmem）
4. 采集磁盘镜像（使用dd/FTK Imager）
5. 导出相关时间段的网络流量和DNS日志
6. 计算所有证据文件的哈希值（SHA256）
7. 填写证据保管链记录表
8. 将证据存储在安全的只读存储中

## 升级标准
- 涉及法律诉讼时通知法务部门
- 证据保全失败时立即报告

## 使用工具
- FTK Imager、Volatility、WinPmem、dd、哈希计算工具
