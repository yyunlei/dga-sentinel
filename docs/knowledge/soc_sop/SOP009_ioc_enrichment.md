# SOP009 - IOC富化流程

## SOP信息
- **编号**: SOP009
- **标题**: DGA相关IOC富化流程
- **触发条件**: DGA告警产生新的IOC（域名、IP、哈希）需要富化

## 响应步骤
1. 提取告警中的IOC（域名、解析IP、关联文件哈希）
2. 查询威胁情报平台（MISP、OTX、VirusTotal）
3. 查询被动DNS数据库获取历史解析记录
4. 查询WHOIS信息获取注册者信息
5. 关联内部SIEM日志查找相关活动
6. 汇总富化结果更新IOC数据库
7. 将高置信度IOC推送到防护设备

## 升级标准
- 发现与APT组织关联的IOC时立即升级
- IOC关联到内部敏感系统时通知安全主管

## 使用工具
- 威胁情报平台、被动DNS、WHOIS、SIEM、IOC管理平台
