# SOP030 - DGA告警分诊（补充）

## SOP信息
- **编号**: SOP030
- **标题**: DGA告警快速分诊检查表
- **触发条件**: 收到任何DGA检测告警

## 响应步骤
1. 检查告警置信度分数（>90%高、70-90%中、<70%低）
2. 查询域名是否在已知白名单中
3. 确认DGA家族分类（已知家族vs未知模式）
4. 检查源主机是否为已知资产
5. 查询域名的VirusTotal/被动DNS记录
6. 检查是否有关联的EDR告警
7. 根据分诊结果分配优先级（P1/P2/P3）
8. 按优先级执行对应SOP

## 升级标准
- P1告警：立即执行SOP006
- 无法判断真假时按真实告警处理

## 使用工具
- DGA检测平台、SIEM、VirusTotal、EDR
