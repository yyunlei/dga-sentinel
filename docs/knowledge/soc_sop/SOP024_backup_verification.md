# SOP024 - 备份验证

## SOP信息
- **编号**: SOP024
- **标题**: DGA感染后备份验证流程
- **触发条件**: DGA感染事件后需要验证备份完整性（尤其是勒索软件相关）

## 响应步骤
1. 确定受影响系统的备份策略
2. 检查最近的备份时间点
3. 验证备份数据完整性（哈希校验）
4. 确认备份数据未被加密或篡改
5. 在隔离环境中测试备份恢复
6. 确定安全的恢复时间点（感染前）
7. 制定恢复计划
8. 执行恢复并验证系统正常

## 升级标准
- 备份数据被加密或损坏时立即升级
- 无可用备份时通知业务负责人和管理层

## 使用工具
- 备份管理系统、哈希校验工具、虚拟化平台
