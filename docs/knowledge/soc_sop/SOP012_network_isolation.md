# SOP012 - 网络隔离操作

## SOP信息
- **编号**: SOP012
- **标题**: DGA感染主机网络隔离操作流程
- **触发条件**: 需要对DGA感染主机执行网络隔离

## 响应步骤
1. 确认隔离目标（IP地址、VLAN、子网）
2. 评估隔离影响（业务中断风险）
3. 获取隔离操作授权
4. 在交换机/防火墙上配置隔离规则
5. 通过EDR执行主机级网络隔离
6. 验证隔离效果（确认无外部通信）
7. 保留管理通道用于远程取证
8. 记录隔离时间和操作详情

## 升级标准
- 隔离影响关键业务时需CTO审批
- 大规模隔离（>10台）时启动应急预案

## 使用工具
- 防火墙、交换机、EDR、网络管理平台
