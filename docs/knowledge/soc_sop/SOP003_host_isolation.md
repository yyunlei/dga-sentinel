# SOP-003: 主机隔离操作流程

## 目的
规范对确认感染 DGA 恶意软件的主机实施网络隔离的操作流程。

## 适用范围
经分析确认存在活跃 C2 通信的受感染主机。

## 前置条件
- SOP-001 分诊结果为高危
- 已确认主机存在恶意 DGA 通信行为

## 流程步骤

### 1. 隔离前准备（10 分钟内）
- 记录主机 IP、MAC、主机名、所属网段
- 确认主机业务角色和影响范围
- 通知主机使用者和业务负责人
- 通过 EDR 获取主机当前进程和网络连接快照

### 2. 执行隔离
- 通过 EDR 平台执行网络隔离（保留管理通道）
- 在交换机/防火墙层面阻断主机外网通信
- 保留主机与 SIEM/EDR 管理平台的连接
- 记录隔离操作时间

### 3. 取证采集
- 导出主机 DNS 查询日志
- 采集内存转储用于恶意软件分析
- 保存相关进程的文件样本
- 记录网络连接和可疑外联 IP

### 4. 恢复评估
- 完成恶意软件清除后进行安全扫描
- 确认无残留后向业务负责人申请恢复
- 恢复后持续监控 72 小时
