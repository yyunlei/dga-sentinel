# SOP007 - 大规模DGA爆发响应

## SOP信息
- **编号**: SOP007
- **标题**: 大规模DGA爆发应急响应流程
- **触发条件**: 短时间内（<1小时）多台主机（>10台）触发DGA告警

## 响应步骤
1. 确认告警规模和影响范围
2. 判断是否为同一DGA家族（统一爆发vs多源攻击）
3. 启动应急响应小组，通知安全主管
4. 在网络边界部署临时阻断规则
5. 批量隔离受影响主机
6. 分析传播路径（邮件、漏洞利用、横向移动）
7. 部署全网扫描确认感染范围
8. 协调IT运维进行批量清理

## 升级标准
- 感染主机超过50台时升级至重大安全事件
- 涉及核心业务系统时启动BCP（业务连续性计划）

## 使用工具
- DGA检测平台、SIEM、EDR、防火墙、漏洞扫描器
