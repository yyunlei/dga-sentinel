# SOP-005: 威胁情报富化流程

## 目的
规范对 DGA 告警中涉及的域名、IP 等指标进行威胁情报查询和富化的流程。

## 适用范围
所有 DGA 检测告警中的可疑域名和关联 IP 地址。

## 流程步骤

### 1. 域名情报查询
- 在 VirusTotal 查询域名检测率和关联样本
- 通过 WHOIS 查询域名注册信息（注册时间、注册商）
- 在 PassiveDNS 数据库查询历史解析记录
- 检查域名是否出现在公开威胁情报源（OTX、MISP）

### 2. IP 情报查询
- 查询 IP 地理位置和 ASN 归属信息
- 在威胁情报平台检查 IP 信誉评分
- 查询 IP 关联的其他恶意域名
- 检查 IP 是否属于已知恶意基础设施

### 3. 关联分析
- 将域名与已知 DGA 家族算法进行匹配
- 分析域名注册模式（批量注册、隐私保护）
- 关联同一 C2 基础设施下的其他恶意活动
- 检查是否与近期安全事件或攻击活动相关

### 4. 情报输出
- 更新内部 IoC 数据库
- 生成威胁情报简报附加到告警工单
- 向社区共享新发现的 DGA 模式和 IoC
- 更新检测规则和模型训练数据集
