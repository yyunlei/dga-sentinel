# SOP017 - 通信协议

## SOP信息
- **编号**: SOP017
- **标题**: DGA安全事件通信协议
- **触发条件**: DGA安全事件需要内部/外部沟通

## 响应步骤
1. 确定事件严重等级和通信范围
2. P1事件：15分钟内通知安全主管和CTO
3. P2事件：1小时内通知安全主管
4. P3事件：在日报中汇报
5. 使用加密通信渠道传递敏感信息
6. 不在公开渠道讨论事件细节
7. 外部通信需经公关部门审核
8. 记录所有通信内容和时间

## 升级标准
- 媒体关注时立即通知公关部门
- 监管机构询问时通知法务部门

## 使用工具
- 加密即时通讯、安全邮件、事件管理平台
