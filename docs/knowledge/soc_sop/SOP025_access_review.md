# SOP025 - 访问审查

## SOP信息
- **编号**: SOP025
- **标题**: DGA感染后访问权限审查流程
- **触发条件**: DGA感染事件后需要审查受影响账户的访问权限

## 响应步骤
1. 确定受影响主机上的用户账户
2. 检查账户是否存在凭证泄露风险
3. 强制重置受影响账户密码
4. 审查账户的权限和访问范围
5. 检查是否存在异常登录记录
6. 撤销不必要的权限
7. 启用多因素认证（如未启用）
8. 监控账户后续活动

## 升级标准
- 发现特权账户被入侵时立即升级
- 发现横向移动使用的凭证时扩大审查范围

## 使用工具
- AD管理工具、IAM平台、SIEM、EDR
