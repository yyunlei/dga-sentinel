# SOP021 - 关联分析

## SOP信息
- **编号**: SOP021
- **标题**: DGA告警关联分析流程
- **触发条件**: 需要对多个DGA告警进行关联分析

## 响应步骤
1. 收集同一时间段内的所有DGA告警
2. 按DGA家族、源IP、时间进行分组
3. 分析告警之间的关联关系
4. 检查是否存在共同的感染源
5. 关联其他安全告警（IDS、EDR、邮件安全）
6. 构建攻击时间线和攻击链
7. 评估整体威胁态势
8. 输出关联分析报告

## 升级标准
- 发现协同攻击迹象时升级至重大事件
- 关联到APT活动时通知威胁情报团队

## 使用工具
- SIEM关联引擎、威胁情报平台、可视化分析工具
