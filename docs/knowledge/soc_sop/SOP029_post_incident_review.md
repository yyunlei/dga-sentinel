# SOP029 - 事后复盘

## SOP信息
- **编号**: SOP029
- **标题**: DGA安全事件事后复盘流程
- **触发条件**: DGA安全事件处置完成后（72小时内）

## 响应步骤
1. 召集事件相关人员参加复盘会议
2. 回顾事件完整时间线
3. 分析检测效果（发现时间、误报/漏报情况）
4. 评估响应效率（响应时间、遏制时间、恢复时间）
5. 识别流程中的不足和改进点
6. 制定改进措施和责任人
7. 更新相关SOP和检测规则
8. 输出事后复盘报告

## 升级标准
- 发现系统性缺陷时制定专项改进计划
- 复盘结果需向管理层汇报

## 使用工具
- 事件管理平台、会议系统、文档协作工具
