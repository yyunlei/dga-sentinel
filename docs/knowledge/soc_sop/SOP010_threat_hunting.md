# SOP010 - DGA威胁狩猎

## SOP信息
- **编号**: SOP010
- **标题**: DGA相关威胁狩猎流程
- **触发条件**: 定期执行（每周）或收到新DGA家族情报时

## 响应步骤
1. 收集最新DGA家族情报和检测规则
2. 在DNS日志中搜索匹配DGA模式的查询
3. 分析NXDomain响应比率异常的主机
4. 检查DNS查询熵值异常的域名
5. 关联EDR数据查找可疑进程
6. 对发现的可疑活动创建告警工单
7. 更新威胁狩猎知识库和检测规则

## 升级标准
- 发现未被检测系统捕获的DGA活动时立即升级
- 发现新型DGA家族时通知威胁情报团队

## 使用工具
- SIEM、DNS日志分析、EDR、DGA检测平台、Jupyter Notebook
