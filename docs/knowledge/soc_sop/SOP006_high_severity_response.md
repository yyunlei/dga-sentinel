# SOP006 - 高严重性DGA告警响应

## SOP信息
- **编号**: SOP006
- **标题**: 高严重性DGA告警响应流程
- **触发条件**: DGA检测系统产生高严重性告警（置信度>90%或已知活跃家族）

## 响应步骤
1. 立即确认告警详情（源IP、域名、DGA家族、置信度）
2. 查询威胁情报平台确认域名/IP的恶意性
3. 检查受影响主机的EDR告警和进程信息
4. 评估影响范围（是否有横向移动迹象）
5. 执行网络隔离（参考SOP003）
6. 采集内存和磁盘取证镜像
7. 通知安全主管和相关业务方

## 升级标准
- 确认为活跃C2通信时立即升级至P1
- 发现数据外传迹象时通知数据安全团队
- 涉及关键业务系统时通知业务负责人

## 使用工具
- DGA检测平台、EDR、SIEM、威胁情报平台、取证工具
