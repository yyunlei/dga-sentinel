# SOP020 - 日志分析

## SOP信息
- **编号**: SOP020
- **标题**: DGA相关日志分析流程
- **触发条件**: 需要深入分析DGA相关日志

## 响应步骤
1. 确定分析目标和时间范围
2. 收集相关日志（DNS日志、防火墙日志、代理日志、EDR日志）
3. 使用SIEM进行日志关联查询
4. 分析DNS查询模式（频率、域名特征、NXDomain比率）
5. 关联网络流量日志确认C2通信
6. 检查主机日志确认恶意进程
7. 生成分析报告和IOC列表
8. 更新检测规则

## 升级标准
- 发现未被检测的恶意活动时立即升级
- 日志缺失或不完整时通知日志管理团队

## 使用工具
- SIEM、ELK Stack、Splunk、日志分析脚本
